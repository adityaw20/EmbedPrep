# Contributing to EmbedPrep

Thank you for your interest in contributing to EmbedPrep! We welcome contributions from the community.

## How to Contribute

### Reporting Issues

If you find a bug or have a suggestion:

1. Check if the issue already exists in the [Issues](https://github.com/yourusername/EmbedPrep/issues)
2. If not, create a new issue with:
   - Clear title and description
   - Steps to reproduce (for bugs)
   - Expected vs actual behavior
   - Screenshots if applicable

### Adding Questions

You can add questions via:

1. **Admin Panel**: Log in and use "Add Question" form
2. **Bulk Upload**: Prepare CSV/JSON and upload
3. **Pull Request**: Add questions to `backend/utils/seed.js`

Question Quality Guidelines:
- Questions should be clear and unambiguous
- Include proper explanations
- Tag appropriately
- Verify correctness of answers

### Code Contributions

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Write/update tests if applicable
5. Submit a pull request

### Development Setup

See [README.md](README.md) for setup instructions.

### Code Style

- Use ESLint and Prettier configurations
- Follow existing code patterns
- Write meaningful commit messages
- Add comments for complex logic

## Code of Conduct

- Be respectful and inclusive
- Constructive criticism is welcome
- Focus on what's best for the community

Thank you for contributing!
